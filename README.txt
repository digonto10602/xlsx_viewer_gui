Xlsx Row Viewer

What it does
- Select an .xlsx file (file picker opens if you run the app without arguments)
- Choose a sheet from the dropdown
- Browse rows with Left/Right keys or Prev/Next buttons
- URLs can be opened with the Open URL button
- Any value can be selected with the mouse and copied:
  - Right click -> Copy
  - or Ctrl+C after selecting text
Then paste anywhere (email, browser, Word, etc).

Right click integration
- After installation, right click any .xlsx file and choose:
  Open with Xlsx Row Viewer
